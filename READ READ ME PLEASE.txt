Como instalar:
crear un archivo solo llamada .env aqui ira tu api de chatgpt en el siguiente formato:
"
OPENAI_API_KEY= tu api de open api
"

abrir terminal dentro de visual studio en una computadora con python instalado

ir a la directoria del proyecto:
  cd c\users\directoria del proyecto ejemplo

instalar requerimientos de python
  pip install -r requirements.txt

dejar la instalacion haga su cosa

ESTE PROYECTO SOLO FUNCIONA EN PYTHON 3.11 o 3.12!!!!!!
